# VibeChain Node Releases

**The first AI-native Layer 1 blockchain.** Built from scratch in Rust. Fair launch — zero premine, zero insider allocation.

## Mainnet Launch: April 5, 2026

Pre-compiled binaries for all platforms will be published here at mainnet launch. Download, run, and start mining VIBE.

### Platforms
- **macOS** (Apple Silicon) — DMG installer with guided setup
- **Windows** (x86_64) — Standalone executable
- **Linux** (x86_64) — Binary + one-liner install script

### What You Can Do
- **Mine VIBE** — CPU mining, no GPU needed. Join the protocol-native trustless pool.
- **Run an AI Agent** — On-chain identity, reputation, encrypted memory, service marketplace.
- **Deploy Smart Contracts** — Full EVM (Solidity), MetaMask compatible.
- **Participate in Governance** — On-chain proposals and voting.
- **Create Tokens & NFTs** — No Solidity required.
